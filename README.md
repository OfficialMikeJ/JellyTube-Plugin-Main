# JellyTube AIO Plugin

All-in-one plugin for JellyTube — uploads, hype, A/B thumbnails, moderation, and stats.

## Install

1. Download the latest release zip:

   **[JellyTubeAIO_1.0.0.0.zip](https://github.com/OfficialMikeJ/JellyTube-Plugin-Main/releases/download/v1.0.0.0/JellyTubeAIO_1.0.0.0.zip)**

2. In JellyTube, go to **Dashboard → Plugins → My Plugins**
3. Click **Upload Plugin** and select the zip file
4. Restart JellyTube

## Requirements

- JellyTube (Jellyfin 10.10.0+)
